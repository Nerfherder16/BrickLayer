### What mem0 Uses by Default

From docs.mem0.ai/components/embedders/overview (scraped 2026-03-15):

> "If no configuration is supplied, a default configuration will be applied, and `OpenAI` will be used as the embedding model."

### Quality Gap for Factual Short-Text Retrieval

The MTEB Retrieval subtask is the most directly relevant metric for memory system use cases (short-text facts stored and retrieved by semantic similarity).

Qwen3-Embedding-0.6B Retrieval scores:
- MTEB English v2 Retrieval: **61.83**
- MTEB Multilingual Retrieval: **64.64**

BGE-M3 Retrieval: 54.60 (multilingual)
nomic-embed-text-v1.5: ~53 (BEIR TREC-COVID top-5 hit rate 86.2% in one test, but lower MTEB composite)

From baseten.co/blog/the-best-open-source-embedding-models/ (Sep 2025): "The best all-around embedding model: Qwen3 8B Embedding" — they note the 0.6B is the right choice when VRAM is constrained.

The step-up from 0.6B to 8B on English Retrieval: 61.83 → 69.44 = **+7.6 points**. This is a real gap but requires ~10x more VRAM (RTX 3090 with 24GB could run 8B in fp16 at ~16GB, leaving only 8GB headroom).

## Mitigation Recommendation

**Low effort, High impact**: Add configurable embedding model support so power users can swap to Qwen3-Embedding-4B or 8B for better retrieval accuracy without architecture changes. The RTX 3090 in Tim's homelab can run 4B in fp16 (~8GB VRAM) comfortably alongside qwen3:14b if batched separately. The 4B model scores 68.46 on English Retrieval vs 61.83 for 0.6B — a +6.6 point improvement that would make Recall's retrieval quality world-class for self-hosted systems.

**Medium effort**: Add support for instruction-aware query prompts. Qwen3-Embedding already supports this natively ("Instruct: ... \nQuery: ...") and the model card states it improves retrieval 1-5%. Recall's current usage should be verified to use the recommended instruct format on the query side.

**Research only**: No immediate action needed on current model. Recall's embedding choice (qwen3-embedding:0.6b) is well-validated and ahead of what most competitors use locally.
