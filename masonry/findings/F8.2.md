## Summary

`_tool_masonry_drift_check()` in `mcp_server/server.py` was modified to use `str(_REPO_ROOT / "agent_db.json")` as the default for `agent_db_path`, matching the same pattern used by `registry_path`. The error-early-return guard `if not agent_db_path_str: return {"error": "agent_db_path is required", ...}` was removed. The tool schema `"required": ["agent_db_path"]` was also removed. Kiln and Claude Code can now invoke `masonry_drift_check` without supplying any arguments.

## Changes Applied

### `_tool_masonry_drift_check()` — default added, guard removed

**Before:**
```python
def _tool_masonry_drift_check(args: dict) -> dict:
    agent_db_path_str = args.get("agent_db_path", "")
    registry_path_str = args.get(
        "registry_path", str(_REPO_ROOT / "masonry" / "agent_registry.yml")
    )

    if not agent_db_path_str:
        return {"error": "agent_db_path is required", "reports": []}

    agent_db_path = Path(agent_db_path_str)
```

**After:**
```python
def _tool_masonry_drift_check(args: dict) -> dict:
    agent_db_path_str = args.get(
        "agent_db_path", str(_REPO_ROOT / "agent_db.json")
    )
    registry_path_str = args.get(
        "registry_path", str(_REPO_ROOT / "masonry" / "agent_registry.yml")

    agent_db_path = Path(agent_db_path_str)
```
