# Finding: D24.3 — `detect_stale_registry_entries()` false positives on Windows due to missing `expanduser()`

**Question ID**: D24.3
**Status**: DONE
**Verdict**: CONFIRMED
**Severity**: Medium
**Confidence**: 0.99
**Agent**: diagnose-analyst

## Summary

`detect_stale_registry_entries()` line 106 calls `Path(file_val).exists()` without `.expanduser()`, causing all 14 `~/.claude/agents/*.md` entries to be falsely reported as stale on Windows.

## Root Cause

File: `masonry/scripts/onboard_agent.py`, line 106.

```python
if file_val and not Path(file_val).exists():
```

On Windows, `Path("~/.claude/agents/mortar.md").exists()` returns `False` because Windows does not expand `~` automatically. The tilde is treated as a literal directory name. `Path(...).expanduser()` resolves `~` to `C:\Users\trg16` and correctly finds the file.

## Evidence

Tested against the live registry (`masonry/agent_registry.yml`):

- **Total agents in registry**: 65
- **Entries with `~` in file path**: 14
- **False-positive stale (raw `Path.exists()` = False)**: 14/14
- **After `expanduser()` (correct result)**: 0/14 stale

All 14 affected entries: test-writer, mortar, spec-writer, developer, karen, prompt-engineer, refactorer, self-host, architect, devops, security, spreadsheet-wizard, rust-analyst, rust-developer.

The `onboard()` function at line 485 calls `detect_stale_registry_entries()` and reports the count in its return dict. This means every full onboard run incorrectly reports 14 stale entries.

## Fix Specification

- **File**: `masonry/scripts/onboard_agent.py`
- **Line**: 106
- **Change**: Replace `Path(file_val).exists()` with `Path(file_val).expanduser().exists()`
- **Verification**: `python -c "from pathlib import Path; assert Path('~/.claude/agents/mortar.md').expanduser().exists()"`
- **Risk**: None. `expanduser()` is a no-op when the path does not contain `~`, so non-tilde paths are unaffected.

## Fix

Change line 106 from:

```python
if file_val and not Path(file_val).exists():
```

to:

```python
if file_val and not Path(file_val).expanduser().exists():
```
