## Summary

`_build_qid_to_agent_map()` in `training_extractor.py` was extended to capture question text from the `### QID: <text>` header in questions.md. `extract_finding()` was extended to extract `**Confidence**:` values from finding files. `extract_training_data()` was updated to populate `finding["question_text"]` from the map. `build_dataset()` now uses `finding.get("question_text") or finding.get("question_id", "")` and `str(finding.get("confidence", 0.75))` instead of hardcoded values. The three degenerate inputs identified in R7.1 are fixed: `question_text` is now the actual question, `confidence` is per-finding, and `project_context` remains `""` pending R8.2 assessment.

## Changes Applied

### 1. `_build_qid_to_agent_map()` — return type changed

**Before:**
```python
def _build_qid_to_agent_map(questions_md_path: Path) -> dict[str, str]:
    ...
    result: dict[str, str] = {}
    blocks = re.split(r"\n---\n", text)
    for block in blocks:
        qid_m = re.search(r"###\s+(\w+\d+\.\d+):", block)
        agent_m = re.search(r"\*\*(?:Agent|Method)\*\*:\s*(\S+)", block)
        if qid_m and agent_m:
            result[qid_m.group(1)] = agent_m.group(1)
    return result
```

**After:**
```python
def _build_qid_to_agent_map(questions_md_path: Path) -> dict[str, dict[str, str]]:
    ...
    result: dict[str, dict[str, str]] = {}
    blocks = re.split(r"\n---\n", text)
    for block in blocks:
        qid_m = re.search(r"###\s+(\w+\d+\.\d+):\s*(.+)", block)
        agent_m = re.search(r"\*\*(?:Agent|Method)\*\*:\s*(\S+)", block)
        if qid_m and agent_m:
            result[qid_m.group(1)] = {
                "agent": agent_m.group(1),
                "question_text": qid_m.group(2).strip(),
            }
    return result
```

The `###\s+(\w+\d+\.\d+):\s*(.+)` regex captures group 2 — the question text after `QID: `. For example:
- `### R7.1: Does \`build_dataset()\` produce...` → `"Does \`build_dataset()\` produce..."`
- `### F8.1: Fix \`build_dataset()\` to extract...` → `"Fix \`build_dataset()\` to extract..."`

### 2. `_CONFIDENCE_RE` added

```python
_CONFIDENCE_RE = re.compile(r"^\*\*Confidence\*\*:\s*(\S+)", re.MULTILINE)
```

### 3. `extract_finding()` — confidence extraction added

```python
conf_m = _CONFIDENCE_RE.search(text)
try:
    confidence = float(conf_m.group(1)) if conf_m else 0.75
except ValueError:
    confidence = 0.75
```

Returns `"confidence": confidence` (float) in the finding dict.

### 4. `extract_training_data()` — question_text population

```python
def _add(finding: dict[str, Any] | None) -> None:
    if finding is None:
        return
    if qid_map and not finding.get("agent"):
        entry = qid_map.get(finding.get("question_id", ""))
        if entry:
            finding["agent"] = entry["agent"]
            if not finding.get("question_text"):
```
