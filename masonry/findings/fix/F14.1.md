## Summary

`bl/nl_entry.py` `_question_to_md()` updated to produce BL 2.0-compatible questions.md blocks. Three field names corrected: `**Mode**:` → `**Operational Mode**:`, `**Test**:` → `**Method**: {agent}`, `**Verdict threshold**:` → `**Success criterion**:`. Header changed from `## ID [PENDING] title` to `### ID: title`. Priority uppercased. New `_MODE_TO_METHOD_AGENT` dict maps mode → specialist agent name.

## Change Applied

**File**: `bl/nl_entry.py`

### Before
```python
def _question_to_md(q: dict) -> str:
    lines = [
        f"## {q['id']} [PENDING] {q['title']}",
        f"**Mode**: {q['mode']}",
        ...
        "**Test**: Read the relevant code and verify the behavior described above.",
        "**Verdict threshold**:",
        "- HEALTHY: ...",
        "- FAILURE: ...",
    ]
```

### After
```python
_MODE_TO_METHOD_AGENT: dict[str, str] = {
    "diagnose": "diagnose-analyst",
    "validate": "benchmark-engineer",
    "research": "research-analyst",
}

def _question_to_md(q: dict) -> str:
    mode = q.get("mode", "diagnose")
    method_agent = _MODE_TO_METHOD_AGENT.get(mode, "research-analyst")
    priority = str(q.get("priority", "medium")).upper()
    lines = [
        f"### {q['id']}: {q['title']}",
        "",
        "**Status**: PENDING",
        f"**Operational Mode**: {mode}",
        f"**Priority**: {priority}",
        "**Source**: nl_entry",
        f"**Hypothesis**: {q['question']}",
        f"**Method**: {method_agent}",
        "**Success criterion**: ...",
    ]
```
