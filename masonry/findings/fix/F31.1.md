## Summary

MIPROv2 output was written only to `optimized_prompts/{agent}.json` and never read at agent spawn time. Claude Code spawns agents by reading their `.md` system prompt files — the JSON was an unreachable artifact.

The fix adds a write-back step: after `optimize_agent()` completes, `run_optimization.py` reads `predict.signature.instructions` from the saved JSON and injects it into every agent `.md` file under a `## DSPy Optimized Instructions` / `<!-- /DSPy Optimized Instructions -->` delimited block. On subsequent runs the block is replaced in-place (idempotent).
