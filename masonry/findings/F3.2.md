## Changes Applied

### Added: `TIER1_TIER2_PATTERNS` and `isTier1Tier2()` function (lines 92-106)

```javascript
const TIER1_TIER2_PATTERNS = [
  /project-brief\.md$/i,
  /agent_registry\.yml$/i,
  /hooks\.json$/i,
  /constants\.py$/i,
  /(?:^|[/\\])src[/\\]/i,    // F2.2: (?:^|[/\\]) catches relative paths
  /(?:^|[/\\])docs[/\\]/i,   // F2.2: (?:^|[/\\]) catches relative paths
];

function isTier1Tier2(filePath) {
  if (!filePath) return false;
  return TIER1_TIER2_PATTERNS.some((p) => p.test(filePath));
}
```

**Pattern analysis:**
- Patterns 1-4 (file suffix): match by filename suffix â†’ catch all path formats (relative, absolute, traversal)
- Pattern 5-6 (directory): `(?:^|[/\\])src[/\\]` â€” `^` catches `src/hooks/foo.js` (starts at root); `[/\\]` catches `./src/`, `C:\path\src\`, `../masonry/src/`

### Added: Guard checks in `main()` (lines 166-174)
