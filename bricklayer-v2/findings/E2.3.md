## Root Causes Fixed (Summary from E2.2)

1. **Training data self-reference**: `files_modified` captured karen's own output files, not trigger source files → model learned "docs present → skip". Fixed: use parent commit source files.

2. **Wrong metric labels for bot commits**: 237/321 records labeled "updated" for "chore: update CHANGELOG" bot commits → systematic mislabeling. Fixed: `_RE_BOT_COMMIT_SUBJECT` pattern in `_derive_expected`.

3. **Windows cp1252 encoding bug in eval harness**: `→` arrows in karen.md truncated the system-prompt argument at the OS level. Fixed: ASCII `->` replacement + stdin-based user_msg passing.

4. **Wrong karen version loaded**: MIPROv2 optimizer overwrote commit-classifier karen.md with project-org version. Fixed: restored from commit c94de5a.
