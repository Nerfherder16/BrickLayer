## Structural Mismatch: Identical to research-analyst

Both agents share the same failure mode:
- **Production**: agent reads campaign files, code, or session outputs → produces evidence-backed HEALTHY
- **Eval**: agent has no tool access → produces INCONCLUSIVE (more honest!) or wrong confidence → 0.00

The structural mismatch is more severe for synthesizer-bl2 because:
1. **More HEALTHY records**: 5/10 vs research-analyst's proportion — synthesizer by design confirms
   things that worked (campaigns, builds, designs)
2. **Session context dependency**: Q6.x records are campaign session summaries requiring the agent
   to read the actual session findings files to assess quality
3. **E8.3 records**: Added with HEALTHY expected for working systems — same trap
