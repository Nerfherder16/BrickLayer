## Pattern Analysis

**Consistently passing records** (expected matches agent's natural output):
- Concrete evaluation questions with known-wrong behavior → WARNING (pilot-1, rec-5)
- Questions with genuinely ambiguous outcomes → INCONCLUSIVE (rec-8)
- Questions with clear falsifiable failures → FAILURE (E9.1-rec-1 new)
- Specific actionable task descriptions with known outcomes → HEALTHY (Q4.4)

**Consistently failing records** (mismatch between expected and natural output):
- Q4.x task descriptions → agent says INCONCLUSIVE (4/5 fail)
- "Is X sound?" HEALTHY questions → agent says INCONCLUSIVE or WARNING
- PROMISING questions → agent says INCONCLUSIVE (lacks bullish framing)
