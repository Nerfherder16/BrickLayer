## Simulation Setup

**Baseline** (nominal recalibrated parameters, Q7.1):
- WAVE_COUNT=4, QUESTIONS_PER_WAVE=7, DOMAIN_NOVELTY=0.35, AGENT_SPECIALIZATION_RATIO=0.65
- yield=0.750, total_questions=28, total_actionable=21
- Base yield per question: 0.750

**Model**: Freed slots are refilled at baseline productivity.
```
additional_actionable = floor(q_b × freed_fraction × base_yield)
new_yield = (a_b + additional_actionable) / q_b
delta = new_yield - yield_b
```

## Comparison to Q6.7

Q6.7 Change 4B modeled DEPLOYMENT_BLOCKED only using a different approach (total_questions *= 0.94 denominator reduction) and reported delta=+0.035. The Q7.4 model uses the refill-at-baseline-productivity approach:

- Q6.7 delta (DB only, denominator reduction): **+0.035**
- Q7.4 Config A (DB only, refill model): **+0.1071**

The 3x difference reflects methodology: Q6.7's denominator-reduction model counts fewer questions while keeping the same actionable count, producing a modest numerator/denominator ratio change. Q7.4's refill model assumes the freed slots are filled with properly-scoped questions that produce findings at the baseline rate — a more optimistic but more realistic model of how campaign managers would actually use the freed capacity.

## Context: Why These Freed Fractions

- **DEPLOYMENT_BLOCKED (15%)**: Q6.3 estimated 12–14 question slots per campaign are blocked by deployment status questions that cannot generate actionable findings. At QPW=7, WAVE_COUNT=4 (28 total), 15% = 4.2 questions → floor(4.2) = 4 questions freed. Used 15% as the mid-estimate.
- **PENDING_EXTERNAL (12%)**: Q6.6 estimated 10–12 slots recovered. At 28 total, 12% = 3.4 questions → floor(3.4) = 3 freed. Used 12% as the mid-estimate.
- **Overlap**: These categories target structurally different question types (deployment-state vs. external-dependency-wait), so overlap is minimal. Config C uses 27% (non-overlapping).

## Interpretation

### Compound effect is additive, not synergistic

Config C delta (+0.1786) equals the arithmetic sum of A+B (+0.1786), ratio = 1.000. The two freed-slot mechanisms operate on independent question slots — there is no interaction effect. This simplifies projection: the total capacity recovery can be estimated by summing individual mechanism contributions.

### Refill quality is the dominant assumption

Both configs assume freed slots are refilled at the baseline yield rate (0.750). If refilled questions are better-scoped (higher novelty, cleaner hypothesis), the actual delta could exceed this estimate. If the refill quality is lower (rushed replacements, lower specialist coverage), the delta shrinks proportionally.

At refill quality 0.50× baseline: Config C delta ≈ +0.089 (still HEALTHY ≥ 0.060).
At refill quality 0.25× baseline: Config C delta ≈ +0.045 (WARNING territory).

### Verdict assessment

The combined delta of +0.1786 clears the HEALTHY threshold (≥0.060) by a wide margin — nearly 3x. Even at 50% refill quality, the compound effect remains HEALTHY. The two mechanisms together represent a meaningful campaign quality lever: eliminating scope-invalid questions and properly filling those slots can push campaign yield from 0.750 to 0.929.
