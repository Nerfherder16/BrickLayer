## Key observations

**Why the current model fails structurally**: `_wave_uniqueness()` implements a monotone decay with a floor of 0.10, reaching that floor at wave 7 (`1.0 - 6 × 0.15 = 0.10`). For any campaign longer than 7 waves, every subsequent wave is assigned 0.10 uniqueness regardless of actual observed signal. The Recall campaign ran 36 waves and showed phases 2 and 3 at 64–66% signal — roughly 6.5x higher than what the model predicts for those waves.

**Model B is better-fit than Model A for two reasons**:
1. RMSE 0.077 vs. 0.143 — Model B is nearly 2x more accurate
2. Model B errors are systematic and small (+5%, +9%, +8.5% per phase) — a single global scale adjustment (multiply all by ~0.90) would reduce RMSE further. Model A overshoots all phases by ~14% uniformly, and its sigmoid inflection at wave 8 produces values well above empirical Phase 1 observations.

**Model B Phase 1 overestimate**: Model B predicts 0.20 for waves 1–7 vs. empirical 0.15. This reflects the hypothesis that early waves are lower-signal than the model's floor. A simple fix: lower the Phase 1 plateau from 0.20 to 0.15 and the Phase 2+ plateau from 0.75 to 0.68, which would drive RMSE below 0.02.

**Suggested refined Model B**:
```python
def wave_uniqueness_jcurve(wave):
    if wave <= 7:
        return 0.15                              # match empirical Phase 1
    elif wave <= 15:
        return 0.15 + 0.53 * (wave - 7) / 8.0   # ramp to ~0.68
    else:
        return 0.68                              # match empirical Phase 2/3 plateau
```
Estimated RMSE with this tuning: ~0.01–0.02.

def _wave_uniqueness(wave: int) -> float:
    """J-curve model: low early signal, rapid ramp, stable late plateau.

    Calibrated to Recall campaign empirical data (36 waves):
      Phase 1 (w1-7):   ~15% signal (low early — agents re-covering familiar ground)
      Phase 2 (w13-24): ~64% signal (high mid — hypothesis generator finds novel space)

    else:
        return 0.68
