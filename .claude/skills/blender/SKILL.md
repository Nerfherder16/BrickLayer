# Blender MCP

## Prerequisites
1. Blender must be running
2. Addon installed: github.com/ahujasid/blender-mcp/blob/main/addon.py
3. Click "Start MCP Server" in Blender sidebar

## Use Cases
- Design 3D printable cases/enclosures
- Create simple 3D models via prompts
- Modify existing Blender scenes
- Export models for printing

## Notes
- Uses uvx at C:\Users\trg16\.local\bin\uvx.exe
- Connection issues: restart both Claude and Blender server
- First command may fail, subsequent ones work
