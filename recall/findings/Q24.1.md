## Finding 1: Code Inspection — Fix Not Deployed

**Location**: `src/storage/qdrant.py:106-114`

```python
def _user_conditions(self, user_id: int | None) -> list:
    """Build Qdrant filter conditions for user_id isolation.

    user_id=None or user_id=0 → no filter (system/admin sees everything).
    user_id>0 → restrict to that user's data.
    """
    if user_id is not None and user_id > 0:
        return [FieldCondition(key="user_id", match=MatchValue(value=user_id))]
    return []
```

`_user_conditions(None)` returns `[]` — no filter applied. The system run at `decay.py:294` calls `worker.run(user_id=None)` which uses this function, resulting in decaying all active memories for the second time per slot.

**Location**: `src/workers/decay.py:292-297`
```python
# Also run for system/unowned memories
try:
    stats = await worker.run(user_id=None)  # ← processes ALL memories, not just unowned
    all_stats["total_decayed"] += stats.get("decayed", 0)
except Exception as e:
    logger.error("decay_system_error", error=str(e))
```

Comment says "system/unowned memories" but behavior processes everything. **No fix deployed.**
