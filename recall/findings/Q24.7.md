## Finding 1: Code Inspection — No IS NULL Filter

**Location**: `src/storage/qdrant.py:1143-1162`

```python
async def get_distinct_user_ids(self) -> list[int]:
    """Return all unique user_id values from the main collection."""
    user_ids: set[int] = set()
    offset = None
    while True:
        points, next_offset = await self.client.scroll(
            collection_name=self.collection,
            limit=250,
            offset=offset,
            with_payload=["user_id"],
            with_vectors=False,
            # ← NO filter here — scans ALL 21,043 points
        )
        for point in points:
            uid = (point.payload or {}).get("user_id")
            if uid is not None:
                user_ids.add(int(uid))
        if next_offset is None:
            break
        offset = next_offset
    return sorted(user_ids)
```

No `scroll_filter` is passed. The scroll iterates all 21,043 Qdrant points (active + superseded), collecting user_ids from superseded memories whose only active memories have since been consolidated away. These ghost user_ids produce processed=0 decay audit entries per slot.

**The fix**: Add a `scroll_filter` with `IsNullCondition(is_null=PayloadField(key="superseded_by"))` to restrict to active memories only.
