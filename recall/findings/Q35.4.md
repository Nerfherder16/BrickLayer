## Required Changes vs Actual State

### Fix 1: `get_distinct_user_ids()` — filter `uid > 0` (`src/storage/qdrant.py:1187`)

**Required**: `if uid is not None and uid > 0:` — excludes the system user (uid=0) from the per-user loop so system memories are not double-decayed.

**Actual** (`qdrant.py:1206–1208`):
```python
for point in points:
    uid = (point.payload or {}).get("user_id")
    if uid is not None:
        user_ids.add(int(uid))
```

The filter is `if uid is not None:` — **no `> 0` check**. If any memory has `user_id=0`, uid=0 will be included in the returned list and decay will run for uid=0 in the per-user loop AND again in the explicit system block. This is the double-decay condition.

**Status: NOT DEPLOYED**
