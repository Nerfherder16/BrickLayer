# Q4.1 — Security Hardening: Path Traversal, CORS, and Hardcoded Base Path

**Verdict**: HEALTHY
**Severity**: Critical (pre-fix) → Resolved
**Commit**: 3425225197cc519554af20b9d1692fc71c924703

## Question

Can the three critical security issues identified in Q3.1 and Q3.2 be fixed with minimal diff and no logic changes?

## Fixes Applied

### Issue 1: Path traversal via `finding_id` (Q3.1 — was FAILURE)

`get_finding()` and `correct_finding()` both used `finding_id` directly in a filesystem path with no containment check. A LAN attacker could supply `../../etc/passwd` as the finding ID.

Fixed by adding `.resolve()` + `startswith()` guard in both endpoints:
```python
fpath = (project_path / "findings" / f"{finding_id}.md").resolve()
findings_dir = (project_path / "findings").resolve()
if not str(fpath).startswith(str(findings_dir)):
    raise HTTPException(status_code=400, detail="Invalid finding ID")
```

### Issue 2: Wildcard CORS on mutation endpoints (Q3.2 — was FAILURE)

`allow_origins=["*"]` with `allow_methods=["*"]` permitted cross-origin POST/DELETE mutations from any origin. The server binds to `0.0.0.0`.

Fixed by restricting to localhost only:
```python
allow_origins=["http://localhost:3100", "http://127.0.0.1:3100"],
allow_methods=["GET", "POST", "PUT", "DELETE", "OPTIONS"],
```

### Issue 3: Hardcoded base path (Q2.3 — was FAILURE)

`get_projects()` used a literal `Path("C:/Users/trg16/Dev/autosearch")`. Fixed to use env var with `__file__`-relative fallback:
```python
base = Path(os.getenv("AUTOSEARCH_BASE", str(Path(__file__).parent.parent.parent)))
```

### Additional: `get_project_path` traversal (discovered during fix)

The `project` query parameter was returned as-is with no containment check, allowing any LAN caller to point the server at any absolute path on the filesystem. Added a guard that resolves the supplied path and requires it to be under `AUTOSEARCH_BASE`.

### Pre-existing lint issue fixed

`domain_map` was assigned but never used in `parse_questions()` (F841). Removed as part of the minimal clean-up needed to pass the lint hook.
