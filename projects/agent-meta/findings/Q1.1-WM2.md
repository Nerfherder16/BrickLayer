# Finding: Q1.1-WM2 — Mortar's `research` routing is ambiguous but the distinction is recoverable from agent descriptions

**Question**: Does Mortar's `research` routing entry specify when to use regulatory-researcher vs competitive-analyst, or is the choice arbitrary?
**Verdict**: WARNING
**Severity**: Low

## Evidence

Mortar's routing table entry:
```
| `research` | regulatory-researcher or competitive-analyst |
```

No selection criteria are specified. However, reading the agent descriptions:

**regulatory-researcher.md** (description):
> "Use for all Domain 2 questions involving compliance, licensing, tax treatment, and regulatory classification."

**competitive-analyst.md** (description):
> "Use for Domain 3 questions about economic sustainability, fee competitiveness, participation rates, and comparable system failure modes."

The selection rule is implicit in the BL 1.x domain structure:
- D2 questions → regulatory-researcher
- D3 questions → competitive-analyst

### Why this is acceptable but suboptimal

A competent LLM orchestrating Mortar would read both agent descriptions and select correctly based on question domain. The ambiguity is resolvable at execution time without confusion — the two agents cover non-overlapping topics.

However, this is fragile:
1. If a research question spans both domains (e.g., "What regulatory changes could make our competitive advantage illegal?"), neither agent alone is sufficient
2. The routing rule could be explicit (D2→regulatory, D3→competitive) with no loss of information

### BL 2.0 context

In BL 2.0 with question-designer-bl2, the `Research` mode routes to `research-analyst.md` (a unified agent) — this ambiguity is a BL 1.x-only issue. It's already resolved in the newer architecture.

## Mitigation Recommendation

1. Update Mortar's routing table comment to be explicit:
   ```
   | `research` | regulatory-researcher (D2 questions) or competitive-analyst (D3 questions) |
   ```
2. Low urgency — this is a documentation improvement, not a correctness fix. The LLM can resolve the ambiguity correctly in practice.
