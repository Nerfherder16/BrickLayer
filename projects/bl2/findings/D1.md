# Finding: D1 — _verdict_from_agent_output() ignores all BL 2.0 verdict types

**Question**: `_verdict_from_agent_output()` in `bl/runners/agent.py` only recognizes `HEALTHY`, `WARNING`, `FAILURE`, `INCONCLUSIVE` as valid self-reported verdicts (line 78-82). When a heal-loop agent self-reports `DIAGNOSIS_COMPLETE`, `FIXED`, `FIX_FAILED`, `COMPLIANT`, `NON_COMPLIANT`, `PROMISING`, `BLOCKED`, `CALIBRATED`, or any other BL 2.0 verdict, it falls through to `INCONCLUSIVE`. This means the heal loop can never receive `DIAGNOSIS_COMPLETE` or `FIXED` from its agents, making it structurally broken.
**Verdict**: FAILURE
**Severity**: Critical
**Mode**: agent

## Evidence

`bl/runners/agent.py` lines 78-82:

```python
self_verdict = output.get("verdict", "").upper()
if self_verdict in ("HEALTHY", "WARNING", "FAILURE", "INCONCLUSIVE"):
    return self_verdict

return "INCONCLUSIVE"
```

Only 4 original BL 1.x verdicts are accepted. All 22+ BL 2.0 verdicts (DIAGNOSIS_COMPLETE, FIXED, FIX_FAILED, COMPLIANT, NON_COMPLIANT, PARTIAL, NOT_APPLICABLE, CALIBRATED, UNCALIBRATED, NOT_MEASURABLE, IMPROVEMENT, REGRESSION, IMMINENT, PROBABLE, POSSIBLE, UNLIKELY, OK, DEGRADED, DEGRADED_TRENDING, ALERT, UNKNOWN, PROMISING, BLOCKED, WEAK, SUBJECTIVE, PENDING_EXTERNAL) all fall through to `INCONCLUSIVE`.

The heal loop depends on `diagnose-analyst` returning `DIAGNOSIS_COMPLETE` and `fix-implementer` returning `FIXED`. Both will ALWAYS return `INCONCLUSIVE` from `_verdict_from_agent_output()`, regardless of what the agent actually outputs in its JSON contract. This makes the heal loop structurally broken — it will never advance past the first `if diag_verdict != "DIAGNOSIS_COMPLETE": break` check.

## Mitigation Recommendation

Add all BL 2.0 verdict types to the self-verdict acceptance check in `_verdict_from_agent_output()`. Import the full verdict set from a constants module or define a `_ALL_VERDICTS` frozenset at module level. Replace lines 78-82 with:

```python
_ALL_VERDICTS = frozenset({
    "HEALTHY", "WARNING", "FAILURE", "INCONCLUSIVE",
    "DIAGNOSIS_COMPLETE", "FIXED", "FIX_FAILED",
    "COMPLIANT", "NON_COMPLIANT", "PARTIAL", "NOT_APPLICABLE",
    "CALIBRATED", "UNCALIBRATED", "NOT_MEASURABLE",
    "IMPROVEMENT", "REGRESSION",
    "IMMINENT", "PROBABLE", "POSSIBLE", "UNLIKELY",
    "OK", "DEGRADED", "DEGRADED_TRENDING", "ALERT", "UNKNOWN",
    "PROMISING", "BLOCKED", "WEAK", "SUBJECTIVE", "PENDING_EXTERNAL",
})

self_verdict = output.get("verdict", "").upper()
if self_verdict in _ALL_VERDICTS:
    return self_verdict
```
