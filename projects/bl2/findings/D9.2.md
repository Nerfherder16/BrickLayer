# Finding: D9.2 — campaign.py agent_db recording skips code_audit mode — BL 2.0 agents not tracked

**Question**: Does campaign.py agent_db recording exclude code_audit mode questions, preventing all BL 2.0 agent runs from being tracked?
**Verdict**: FAILURE
**Severity**: Medium
**Mode**: diagnose
**Operational Mode**: diagnose

## Root Cause

`bl/campaign.py` `run_and_record()` lines 156-169:

```python
# BL 2.0: record agent performance in agent_db
agent_name = question.get("agent_name")
if question.get("mode") == "agent" and agent_name:
    try:
        from bl import agent_db
        score = agent_db.record_run(cfg.project_root, agent_name, result["verdict"])
        ...
```

Condition: `question.get("mode") == "agent"` — only records when mode is exactly "agent".

After F5.1 (parse_questions uses body **Mode** field) and F6.1 (code_audit registered as alias for run_agent), BL 2.0 questions have:
- `question["mode"] = "code_audit"` (from body `**Mode**: code_audit`)
- `question["agent_name"]` = agent name (e.g., "diagnose-analyst", "fix-implementer")

The mode is "code_audit", not "agent". The `if mode == "agent"` condition is False. `agent_db.record_run()` is never called.

## Scope of Impact

All 50 BL 2.0 questions in this campaign use `**Mode**: code_audit`. That's:
- 0 agent_db entries for diagnose-analyst
- 0 agent_db entries for fix-implementer
- 0 agent_db entries for compliance-auditor
- 0 agent_db entries for design-reviewer

The overseer underperformer detection cannot fire for any BL 2.0 agent.

## Verification

```bash
ls projects/bl2/agent_db.json 2>/dev/null || echo "not found"
```

Expected: agent_db.json either doesn't exist or has no entries for BL 2.0 agents.

## Fix Specification

- **File**: `bl/campaign.py`, `run_and_record()`, agent_db recording block (line 158)
- **Change**: `if question.get("mode") == "agent" and agent_name` → `if question.get("mode") in ("agent", "code_audit") and agent_name`
- **Verification**: Run a code_audit mode question → agent_db.json contains entry for the agent
