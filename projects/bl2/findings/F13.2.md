```python
# Extract operational_mode from **Operational Mode**: field (added by F13.1)
# and map to a valid BL 2.0 bracket tag.  Fall back to runner-mode mapping,
# then to DIAGNOSE as a safe default.
_OP_MODE_TO_TAG: dict[str, str] = {
    "diagnose": "DIAGNOSE",
    "fix": "FIX",
    "audit": "AUDIT",
    "validate": "VALIDATE",
    "monitor": "MONITOR",
    "frontier": "FRONTIER",

header_match = re.match(r"^## \S+(.*)$", seg, re.MULTILINE)
if header_match and "[" not in header_match.group(1):
    # Prefer **Operational Mode**: field (canonical BL 2.0 field)
    op_mode_match = re.search(
        r"\*\*Operational Mode\*\*:\s*(\w+)", seg, re.IGNORECASE
    )
    if op_mode_match:
        raw_op = op_mode_match.group(1).lower()
        mode_tag = f"[{_OP_MODE_TO_TAG.get(raw_op, 'DIAGNOSE')}]"
    else:
        # Legacy fallback: **Mode**: runner_mode → mapped bracket tag
        mode_match = re.search(r"\*\*Mode\*\*:\s*(\w+)", seg)

            raw_mode = mode_match.group(1).lower()
            mode_tag = f"[{_OP_MODE_TO_TAG.get(raw_mode, 'DIAGNOSE')}]"
        else:
            mode_tag = None
    if mode_tag:
        seg = re.sub(
            r"^(## \S+) ",
            rf"\1 {mode_tag} ",
            seg,
            count=1,
            flags=re.MULTILINE,
        )
```

## Verification

Trace through three cases:

1. **Sub-question with `**Operational Mode**: audit`** (canonical BL 2.0, added by F13.1):
   - `op_mode_match` finds `audit` → `.lower()` → `"audit"` → `_OP_MODE_TO_TAG["audit"]` = `"AUDIT"` → injects `[AUDIT]`. Correct.

2. **Runner mode `code_audit`, no `**Operational Mode**:` field** (legacy fallback path):
   - `op_mode_match` fails → falls to `**Mode**: code_audit` → `"code_audit"` → `_OP_MODE_TO_TAG["code_audit"]` = `"AUDIT"` → injects `[AUDIT]`. Correct. Previously produced `[CODE_AUDIT]` (invalid).

3. **Runner mode `agent`** (BL 1.x legacy):
   - `"agent"` → `_OP_MODE_TO_TAG["agent"]` = `"DIAGNOSE"` → injects `[DIAGNOSE]`. Reasonable fallback.

4. **No mode field at all**:
   - `mode_tag = None` → no injection attempted, header unchanged. Safe.

Syntax check: `python -m py_compile bl/followup.py` → OK.

## Backward Compatibility

- BL 1.x questions with `**Mode**: agent` → `[DIAGNOSE]` — reasonable fallback, matches the general-purpose diagnostic intent of the agent runner.
- Blocks that already have a bracket tag in the header (`[` present) are skipped by the guard `if "[" not in header_match.group(1)` — no double-injection.
- All nine valid BL 2.0 operational modes map 1:1 to their bracket tags.
