## Fix Applied

Added four scorer functions to `bl/crucible.py` and registered them in `_SCORERS`. All follow the `_score_synthesizer()` static-file pattern: read findings/*.md and results.tsv, extract text metrics, return `AgentScore(name, score, checks_raw, notes)`.

### Scorers added:

**`_score_diagnose_analyst`**
- DIAGNOSIS_COMPLETE+HEALTHY rate from results.tsv (weight 0.60)
- Fix-spec completeness fraction: findings with all 4 required fields (Target file, Target location, Concrete edit, Verification command) (weight 0.40)

**`_score_fix_implementer`**
- FIXED rate from fix rows in results.tsv (weight 0.45)
- Verification/Fix Applied/Evidence section presence in FIXED findings (weight 0.35)
- Reliability = 1 - (FIX_FAILED_rate × 2) (weight 0.20)

**`_score_compliance_auditor`**
- Definitive verdict rate (COMPLIANT/NON_COMPLIANT/PARTIAL vs INCONCLUSIVE) (weight 0.60)
- NON_COMPLIANT findings with Fix Specification section (weight 0.40)

**`_score_design_reviewer`**
- COMPLIANT rate from review rows in results.tsv (weight 0.55)
- Line-number reference rate in validate/review findings (weight 0.45)
