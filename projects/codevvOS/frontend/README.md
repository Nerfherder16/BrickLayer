# CodeVV OS — Frontend

React 19 + TypeScript + Tailwind v4 frontend for CodeVV OS.

## Font Setup

Real font files are required before deployment. Run: `make download-fonts`

Place `.woff2` files in `frontend/public/fonts/`:

- `Inter-400.woff2`
- `Inter-500.woff2`
- `Inter-600.woff2`
- `JetBrainsMono-400.woff2`
- `JetBrainsMono-500.woff2`

Download Inter from: https://rsms.me/inter/
Download JetBrains Mono from: https://www.jetbrains.com/lp/mono/
