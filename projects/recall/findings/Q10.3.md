## Verdict Threshold

- FAILURE: mean importance < 0.35 for any memory type
- WARNING: p25 < 0.30 for any type OR >30% of memories at exactly same value
- HEALTHY: mean >= 0.40 across all types AND p25 >= 0.30

## Mitigation Recommendation

**Importance calibration**: Technical domains (documentation, ai-ml, frontend) averaging 0.19–0.22 suggests the signal extraction pipeline undervalues structured/technical content relative to narrative content. Investigate the signal scoring weights in `src/core/signals.py`. The distribution has not improved after 1,835 decay events — decay alone cannot raise the mean; only signal re-scoring or importance backfill can.
