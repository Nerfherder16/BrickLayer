## Test Pattern

```python
detector = SignalDetector.__new__(SignalDetector)
signals = detector._parse_response(json.dumps([{
    "signal_type": "fact",
    "content": "...",
    "confidence": 0.9,
    "importance": 5,
    "durability": "durable",
    "domain": "api",
    "tags": []
}]))
assert signals[0].suggested_importance >= 0.5
```
